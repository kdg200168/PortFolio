こちらは3Dシューティングゲームです。  
Unityを使用して制作しました。  
制作期間は1か月です。  
使用言語はC#です。  
こだわった点は自機の動きです。  
  
操作方法  
マウス操作 クリック操作のみ  
キー操作　WASDキー　自機移動  
        SHIFTキー 加速  
        Zキー　減速  
        SPACEキー 自弾発射
