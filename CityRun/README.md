こちらはパルクールアクションゲームです。  
Unityを使用して制作しました  
制作期間は3か月です。  
こだわった点はUIのアニメーションと壁走りの実装です。  
  
操作方法  
マウス操作  
キー操作　WASDキー　プレイヤー移動  
          SPACEキー ジャンプ,壁走り,壁キック  
          Escキー 一時停止(プレイ画面中のみ)
